# guru_guru

Transforma lo que quieras al lenguaje Chileno

## Instalación

Agrega la siguiente línea en tu Gemfile:

    gem 'to_chilean'

y ejecuta:

    $ bundle

O instala de forma manual:

    $ gem install to_chilean

## Uso

```ruby
require 'to_chilean'

"Hola ¿como estas?".to_chilean => "Ola ¿como estai?"

```
